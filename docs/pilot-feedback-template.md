# Pilot Feedback Template

Use this with the first 1-3 teammates before a wider rollout.

## Quick Context

- Name:
- Role:
- Main surface used first: `Dashboard` / `Reports` / `Settings`
- Date tested:

## Setup Experience

1. How long did setup take from opening the app to a trusted first view?
2. Did the `Setup wizard` and `Diagnostics` make the next step obvious?
3. Did you need a proxy or other extra environment setup?
4. What was the most confusing part of setup?

## Data Trust

1. Did the Jira data shown in `Dashboard` or `Reports` match what you expected?
2. If something looked wrong, which surface showed it?
3. Did the in-app validation or diagnostics help explain the issue?

## Day-To-Day Use

1. Which flow felt most useful?
   - weekly close in `Dashboard`
   - weekly team view in `Reports`
   - monthly reporting in `Reports`
2. Which action felt easiest?
3. Which action felt harder than it should?

## Rollout Readiness

1. Would you feel comfortable using this without pairing next time?
2. Would you recommend it to the rest of the team now?
3. What is the one thing we should fix before broader rollout?
